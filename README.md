# Imagine — Download

AI-powered image & video generation desktop app.

## Download

Go to [Releases](https://github.com/nikola-arman/imagine-releases/releases) and download the latest version for your Mac.

| File | For |
|---|---|
| `Imagine-*-mac-arm64.dmg` | Apple Silicon (M1/M2/M3/M4) |
| `Imagine-*-mac-x64.dmg` | Intel Macs |

## Installation

1. Download the `.dmg` file for your Mac
2. Open it and drag **Imagine** to your Applications folder
3. First launch: **right-click → Open** (to bypass Gatekeeper since the app is unsigned)
4. Create a 4-digit PIN
5. Click the ⚙ gear icon to enter your EternalAI API key

## Features

- 89 AI effects (40 image, 49 video) with one-click apply
- Parallel generation with inline progress
- Video effect cards auto-play
- Full-size preview panel with before/after comparison
- PIN code protection (locks on minimize)
- Dark glassmorphism UI

## Requirements

- macOS 10.15+ (Catalina or later)
- EternalAI API key ([get one here](https://docs.eternalai.org))
